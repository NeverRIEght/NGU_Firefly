# NGU_Firefly

Firefly is a video compression tool, designed to achieve maximum compression efficiency while preserving hardware and software compatibility. Firefly performs per-video analysis to encode H265-based videos with perfect, visually lossless compression. It is created for the people who want to store their videos in long term, while cutting costs of either personal hardware or cloud storage solutions.
